Gemini API Key

Get 5 free searches with our default server key. After that, please enter your own Gemini API Key below.

Your key is stored locally in this browser and never shared.

How to get Gemini API Key

1. Sign In to Google AI Studio

Open your browser and navigate to Google AI Studio.

Log in using your standard Google/Gmail account.

Review and accept the Terms of Service if this is your first time logging into the platform.

2. Create the API Key

Click the Get API key button located on the top-left sidebar dashboard.

Click the Create API key button.

Provide a descriptive name for your key to track your project.

Select an existing Google Cloud Project or click Create project to establish a new one for grouping your keys.

Click Create key to finalise the generation process.

3. Copy and Secure Your Key

Click the Copy icon next to the generated string to save it to your clipboard.
